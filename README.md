# WWALK Lab: Data & GitHub Workshop
Authors: Isabella Richmond  
Date of Workshop: May 19, 2026

The goal of this workshop is to familiarize the WWALK lab with knowledge about creating and contributing to a GitHub repository, best practices for file and project management, and archiving data and code publicly.
